Welcome to Horcrux Password Manager

Set up keystores!

See our README for AWS and Azure account creation details and keystore setup. After completing that, use an offline text editor to format your keystore credentials as a JSON string to use with Horcrux.
The format is as follows: {"accessKeyId1":"actual access key ID goes here","secretAccessKey1":"actual secret key goes here","region1":"one of AWS regions, e.g. eu-central-1","accessKeyId2":"second keystore access id","secretAccessKey2":"another secret access key","region2":"a different jurisdiction"} and so forth.
